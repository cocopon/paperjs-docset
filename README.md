paperjs-docset
==============
Paper.js docset generator for [Dash](http://kapeli.com/dash).




System Requirements
-------------------
- Ruby 2.0 and above
- RubyGems
  - Nokogiri
  - Sqlite3
- Git




Generating Docset
-----------------
1. Install required gems if not installed yet:
```
$ gem install nokogiri sqlite3
```

2. Execute a generator script:
```
$ cd path/to/paperjs-docset
$ ruby docset.rb
```
